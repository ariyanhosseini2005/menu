# 🌿 Sandton Garden — وب‌سایت رستوران سندتن گاردن

وب‌سایت کامل و حرفه‌ای رستوران سندتن گاردن با **بک‌اند Node.js + SQLite**، شامل منوی آنلاین، سفارش، رزرو ۱۴ آلاچیق و سالن، نظرات مهمانان و یک **پنل مدیریت کامل**.

طراحی مدرن با هویت رنگی **زرد و بنفش**، انیمیشن‌های حرفه‌ای، و رابط راست‌به‌چپ (RTL) فارسی.

---

## ✨ امکانات

- **صفحه‌ی اصلی (Home):** هیرو، پیشنهادهای ویژه، معرفی، نظرات و گالری.
- **منو (`/menu.html`):** دسته‌بندی، جستجو، افزودن به سبد، برچسب «پرطرفدار» و قیمت‌ها از بک‌اند.
- **رزرو آلاچیق (`/reserve.html`):** انتخاب تاریخ/ساعت/نفرات، نقشه‌ی ۱۴ آلاچیق + سالن، **جلوگیری از رزرو دوبل در سمت سرور**، «رزروهای من» با امکان لغو.
- **سفارش آنلاین:** سبد خرید سراسری + فرم ثبت سفارش + ارسال به واتساپ.
- **درباره/تماس (`/about.html`):** ساعات کاری، اطلاعات تماس و **ثبت نظر** (با تأیید مدیر).
- **گالری (`/gallery.html`):** نمایش تصاویر با Lightbox.
- **پنل مدیریت (`/admin.html`):** مدیریت کامل منو و قیمت‌ها، رزروها، سفارش‌ها، نظرات، آلاچیق‌ها، تنظیمات سایت و تغییر رمز عبور.

## 🧱 معماری

```
server/
  server.js       # Express app + REST API + سرو فایل‌های استاتیک
  db.js           # SQLite (better-sqlite3): اسکیما، seed و توابع کمکی
  seed-menu.json  # داده‌ی اولیه‌ی منو و تصاویر (استخراج‌شده از نسخه‌ی قبلی)
  smoke-test.js   # تست دود (smoke test)
  data/           # فایل دیتابیس (ساخته‌می‌شود؛ در git نیست)
public/
  index.html, menu.html, reserve.html, about.html, gallery.html, admin.html, 404.html
  assets/css/styles.css        # سیستم طراحی مشترک
  assets/js/common.js          # هدر/فوتر/اسپلش/سبد/افکت‌ها + helper API
  assets/js/{home,menu,reserve,about,gallery,admin}.js
  logo.jpg
```

## 🚀 اجرا

```bash
npm install
npm start            # روی http://localhost:3000
```

برای توسعه با ری‌استارت خودکار:

```bash
npm run dev
```

تست سریع API:

```bash
npm run smoke
```

ساخت دوباره‌ی داده‌ی اولیه (پاک‌سازی و seed مجدد منو/آلاچیق‌ها):

```bash
npm run seed
```

## 🔐 پنل مدیریت

به آدرس `/admin.html` بروید.

- رمز عبور پیش‌فرض: **`sandton1234`**
- می‌توانید رمز را با متغیر محیطی هنگام اولین اجرا تنظیم کنید:

```bash
ADMIN_PASSWORD="رمز-قوی-شما" npm start
```

- پس از ورود، از بخش **«رمز عبور»** حتماً رمز را تغییر دهید.

از پنل می‌توانید:
- **قیمت‌ها و آیتم‌ها** را ویرایش/اضافه/حذف کنید و عکس هر آیتم را تغییر دهید (لینک یا Base64).
- وضعیت **رزروها** و **سفارش‌ها** را مدیریت کنید.
- **نظرات** را تأیید/حذف کنید.
- نام/ظرفیت/فعال‌بودن **آلاچیق‌ها** را تنظیم کنید.
- **تنظیمات سایت** (متن هیرو، تماس، ساعات کاری، واحد قیمت) را ویرایش کنید.

## 🗄️ پایگاه‌داده

از SQLite (فایل `server/data/sandton.db`) استفاده می‌شود و در اولین اجرا به‌صورت خودکار از `seed-menu.json` پر می‌شود. این فایل در `.gitignore` است تا داده‌ی واقعی در مخزن قرار نگیرد.

## 📝 نکته درباره‌ی قیمت‌ها و عکس‌ها

قیمت‌ها و عکس‌ها را می‌توانید مستقیماً از **پنل مدیریت** ویرایش کنید — نیازی به دست‌زدن به کد نیست.

## 🌐 استقرار (Deployment)

این یک اپ Node است؛ روی هر هاستی که Node را اجرا می‌کند (Render, Railway, VPS, …) قابل اجراست:

```bash
npm install --omit=dev
PORT=8080 ADMIN_PASSWORD="..." node server/server.js
```

پیشنهاد: یک reverse proxy (Nginx) جلوی اپ قرار دهید و دامنه‌ی `sandton.garden` را به آن وصل کنید.
